welcome to paLit

Color Palette Creator

paLit allows a user to create custom color palettes for their websites. Simply create a palette, and click on a color to save it's Hex, RGB, or ARGB code to your clipboard. Then simply add it into your project.

I really enjoyed working on paLit. It is a tool that I can use to help with all of my future projects, and I got to use a lot of libraries for the first time, including Chroma-js, Material UI, and react-sortable-hoc.
